# אודות הפרויקט

זהו פרויקט של מערכת מבחנים שפועלת בדפדפן בלבד.

המורים יכולים ליצור ולנהל מבחנים, והסטודנטים יכולים לחפש מבחנים, לבצע אותם ולראות את הציונים שלהם.

הפרויקט נבנה באמצעות JavaScript בלבד, וכל המידע נשמר ב־localStorage.

# דפי המערכת והניווט ביניהם

דף הבית (index.html)
זהו הדף הראשון במערכת.
ממנו ניתן לעבור אל:
•	דף הרשמה 
•	דף התחברות 

דף הרשמה (register.html)
משתמש חדש יכול ליצור חשבון.
ניתן לבחור אם המשתמש הוא:
•	מורה 
•	סטודנט 

דף התחברות (login.html)
משתמש קיים מתחבר באמצעות שם משתמש וסיסמה.
לאחר התחברות:
•	מורה מועבר לדף המורה. 
•	סטודנט מועבר לדף הסטודנט. 


דף המורה (teacher.html)
המורה יכול:
•	ליצור מבחנים 
•	לערוך מבחנים 
•	למחוק מבחנים 
•	לצפות במבחנים שיצר 
•	לעבור לדף פרטי מבחן 

דף פרטי מבחן (exam-details.html)
בדף זה ניתן:
•	להוסיף שאלות 
•	למחוק שאלות 
•	לצפות בתוצאות הסטודנטים 

דף חיפוש מבחן (search.html)
הסטודנט יכול לחפש מבחן לפי:
•	שם מבחן 
•	קוד מבחן 

דף ביצוע מבחן (take-exam.html)
בדף זה הסטודנט:
•	קורא את השאלות 
•	בוחר תשובות 
•	מגיש את המבחן 
•	מקבל את הציון 

דף הסטודנט (student.html)
בדף זה ניתן לראות:
•	מבחנים שבוצעו 
•	ציונים קודמים 
•	ממוצע ציונים 
•	מעבר לחיפוש מבחן חדש

# שמירת הנתונים

כל הנתונים במערכת נשמרים ב־localStorage בפורמט JSON.

הנתונים שנשמרים הם:

* users
* exams
* results
* currentUser

# זרימת העבודה

## זרימת העבודה המרכזית

1. המורה מתחבר למערכת.
2. המורה יוצר מבחן חדש.
3. המורה מוסיף שאלות ושומר את המבחן.
4. הסטודנט מתחבר למערכת.
5. הסטודנט מחפש מבחן לפי שם או קוד.
6. הסטודנט מבצע את המבחן.
7. המערכת מחשבת את הציון באופן אוטומטי.
8. התוצאה נשמרת ב־localStorage.
9. גם הסטודנט וגם המורה יכולים לראות את התוצאה.

---

## התחברות

1. המשתמש פותח את דף ההתחברות.
2. המשתמש מכניס שם משתמש וסיסמה.
3. AuthService בודק את פרטי המשתמש.
4. אם ההתחברות הצליחה, המשתמש נכנס למערכת.
5. המשתמש מועבר לדף המתאים (מורה או סטודנט).

---

## יצירת מבחן

1. המורה נכנס לדף המורה.
2. לוחץ על יצירת מבחן חדש.
3. ממלא את פרטי המבחן.
4. ExamService שומר את המבחן.
5. המבחן מופיע ברשימת המבחנים.

---

## הוספת שאלות

1. המורה פותח מבחן קיים.
2. מוסיף שאלה חדשה.
3. מוסיף את אפשרויות התשובה.
4. בוחר את התשובה הנכונה.
5. השאלה נשמרת כחלק מהמבחן ומתעדכנת ב־localStorage.
